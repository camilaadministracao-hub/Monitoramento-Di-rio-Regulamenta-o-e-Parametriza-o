MONITORAMENTO REGULATÓRIO DIÁRIO — CUSTO ZERO

Arquitetura escolhida: GitHub Pages + GitHub Actions.

Vantagens:
- sem instalação no computador da empresa;
- custo R$ 0,00;
- uso por qualquer colega nas férias;
- backup e histórico de código;
- monitoramento automático pela manhã.

Passos:
1. Criar repositório no GitHub.
2. Enviar todos os arquivos desta pasta.
3. Ativar GitHub Pages.
4. Em Actions, executar manualmente o workflow "Monitoramento Regulatório Diário".
5. Abrir o link do GitHub Pages e clicar em "Carregar monitoramento do dia".

Observação:
O módulo DOU deverá ser evoluído em etapa própria, pois a busca exige filtros específicos.
